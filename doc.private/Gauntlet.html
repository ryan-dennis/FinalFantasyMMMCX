Module Gauntlet

module Gauntlet: sig .. end

Representation of static gauntlet data.

This module represents the data stored in gauntlet files, including all the bosses and their characteristics. It handles loading of that data from a JSON file as well as querying the data.

type boss_id = string

The type of boss.

type sprite = string list

The type of sprite.

exception UnknownBoss of boss_id

Raised when an unknown boss is encountered.

exception InvalidSprite of sprite

Raised when an invalid sprite is encountered.

The type of starting information.

The abstract type of values representing gauntlets.

val start_of_json : Yojson.Basic.t -> start

start_of_json j is the starting information in the gauntlet j represents.

val stats_of_json : Yojson.Basic.t -> stats

stats_of_json j is the stats of a boss in the gauntlet j represents.

val magic_of_json : Yojson.Basic.t -> magic

magic_of_json j is the magic abilities of a boss in the gauntlet j represents.

val special_of_json : Yojson.Basic.t -> special

special_of_json j is the special abilities of a boss in the gauntlet j represents.

val check_sprite : sprite -> sprite

check_sprite sprite is sprite if it is a valid sprite (each line is 30 characters) and raises InvalidSprite otherwise.

val boss_of_json : Yojson.Basic.t -> boss

boss_of_json j is a boss in the gauntlet j represents.

val from_json : Yojson.Basic.t -> t

from_json j is the gauntlet that j represents.

val start_boss : t -> boss_id

start_boss g is the identifier of the starting boss in gauntlet g.

val start_dialogue : t -> string

start_dlg g is the opening dialogue in gauntlet g.

val final : t -> boss_id

final g is the idenitifier of the final boss in gauntlet g.

val find_boss : t -> boss_id -> boss

find_boss glt b is the boss in glt with the identifier b.

val boss_stats : t -> boss_id -> stats

boss_stats g b is the stats of boss b in gauntlet g.

val boss_sprite : t -> boss_id -> string list

boss_sprite g b is the sprite of boss b in gauntlet g.

val boss_spell_chance : t -> boss_id -> int

boss_spell_chance g b is the spell chance of boss b in gauntlet g.

val boss_spells : t -> boss_id -> string list

boss_spells g b is the list of spells of boss b in gauntlet g.

val boss_skill_chance : t -> boss_id -> int

boss_skill_chance g b is the skill chance of boss b in gauntlet g.

val boss_skills : t -> boss_id -> string list

boss_skills g b is the list of skills of boss b in gauntlet g.

val next : t -> boss_id -> string

next g b is the next boss after b in gauntlet g.

val dialogue : t -> boss_id -> string

get_dlg g b is the victory dialogue for boss b in gauntlet g.
