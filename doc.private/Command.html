parse s takes in a string that is either a validly formed command or an invalid command. A command can be invalid in two ways. If the string is empty or a list of spaces it will raise Empty. If the command is a string with characters but is malformed then parse will raise exception Malformed. If the string is a valid command, then the valid command will be returned
